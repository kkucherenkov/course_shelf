## Goal

Issue an opaque token tied to (userId, lessonId, expiresAt), verifiable without DB lookup.

## Acceptance

- Token is HMAC-signed with `BETTER_AUTH_SECRET`-derived subkey
- Default TTL = 15 min
- Verification fails on tamper, expiry, or mismatched lesson
