CourseShelf · Design system v0.1

A self-hosted home
for technical courses.

Plex for technical courses — for people who already have a folder of MP4s and PDFs and want to actually finish them. Calm, dense, keyboard-friendly. Built on a small set of foundations that scale from desktop player to mobile downloads.
