## Notes

- Completed: 2026-04-27
- Result: T-2026-04-27-046 — single commit `1ce6208` on `feat/navigation-shell`. UI tests 707/707 (+45); lint + typecheck clean.
- CSS-only responsive switch (no resize handler). Sidebar nav uses `AppRow` with `interactive: true` + `compact: true` + `selected` when key matches `activeRoute`. `aria-current="page"` falls through Vue's default `inheritAttrs`.
- Avatar menu: shared popover state across two triggers (sidebar user block + topbar avatar). Items are `role="menuitem"` with arrow-key nav between them; closes on ESC, click-outside (document `mousedown` listener), or item activation. Profile / Settings / Sign out emit dedicated events; theme menu item toggles via `update:colorMode`.
- Bottom-tab bar shows the primary `nav` only (no admin), capped at 5 items. `nav.length > 5` truncates and is documented in the prop doc.
- `right-rail` slot is opt-in: when present, `main` becomes `1fr 280px`; when absent, single column.
- `ShellUser.role` (`USER | ADMIN | GUEST`) maps to `AppAvatar.role` via a small computed (`ADMIN → 'admin'`, `GUEST → 'guest'`, `USER → undefined`).
- Routing-agnostic — emits `nav` with the item `key`. Consumers wire to their own router.
