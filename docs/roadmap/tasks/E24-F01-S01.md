## Goal

Realtime spine: clients receive a short-lived Centrifugo JWT (TTL ≤5 min) from `POST /api/v1/realtime/token`; AsyncAPI 3.0 documents the channels they subscribe to.

## Acceptance

- `packages/specs/asyncapi/centrifugo.yaml` describes core channels (e.g. `library:scan`, `notes:lesson:{id}`)
- `POST /api/v1/realtime/token` declared in OpenAPI; returns `{ token, expiresAt }`
- Token TTL ≤5 minutes; signed with `CENTRIFUGO_TOKEN_HMAC` from `AppConfig`
- Endpoint requires authenticated session; per-user `sub` claim in the JWT payload
