## T-2026-04-28-059 — Stage A Lesson player wired to <video> (E14-F03-S01)

- Created: 2026-04-28
- Owner: claude
- Spec: `docs/roadmap/tasks/E14-F03-S01.md` (source: `docs/design/cs-web-lesson-player/`, DESIGN_BRIEF §6.6)
- Goal: ship the real lesson-player page — `AppPlayerChrome` attached to a real `<video>`, progress reporting via `recordLessonProgress`, right-sidebar tabs (Sections / Notes / Bookmarks / Materials) loading content inline. No backend or spec work — every endpoint already exists.
- Acceptance:
  - Route `apps/web/app/pages/courses/[id]/lessons/[lessonId].vue`. Pulls lesson + outline + bookmarks + note in parallel.
  - `<video>` element wired to a stream URL fetched via `issueStreamUrl(lessonId)`. `AppPlayerChrome` is the visual layer; its `play / pause / seek / speed / toggleSubtitles / togglePip / toggleFullscreen / nextLesson / prevLesson / toggleMute` events map to the underlying video element.
  - Progress reporter posts `recordLessonProgress` every 10s of played time + on `visibilitychange` (when the tab hides) + on `beforeunload`. The reporter dedupes — never two in-flight at once.
  - Resume from server progress: `LessonDto.progress.lastSeenAtSeconds` seeds the `<video>.currentTime` after metadata is loaded.
  - Speed picker (0.5 / 1.0 / 1.25 / 1.5 / 1.75 / 2.0) + subtitle picker (off + every available `SubtitleDto`).
  - Auto-advance banner at end-of-lesson (5s countdown → auto-loads next lesson via outline order; `Stay here` cancels).
  - On 401 from the stream URL endpoint: refresh token via the existing auth store hook and retry once.
  - Sidebar tabs (`AppTabs` + `AppTab`):
    - **Sections**: course outline, current lesson highlighted via `AppLessonRow current=true`. Click → navigate to that lesson.
    - **Notes**: `AppNoteEditor` v-modeled to the per-lesson note. `save(value)` → `upsertNote({ lessonId, body })`. Sync indicator reflects `pending → saved → failed → offline`.
    - **Bookmarks**: `AppBookmarkList` with the lesson's bookmarks. Add row's `time` is the current playhead. `addSave` → `createBookmark`; `delete` → `deleteBookmark`. Selecting a bookmark seeks the video.
    - **Materials**: list of `LessonDto.materials` with download links (no-op — emits a "Download coming soon" toast).
  - Layouts: 1440 → player ~70% / sidebar ~30%; 1024 → narrower sidebar; 360 → player full-width on top, tabs as horizontal scroller below.
  - States: Loading (skeleton), Playing, Buffering, Error, EndOfLesson, NoPermission.
- Spec diff: none.
- Codegen impact: none.
- Design impact: none — consumes existing components.
- Tests: vitest unit for `useProgressReporter` (timer dedupe, visibility-change, beforeunload). Playwright e2e: login → open lesson → seek → close → reopen → resume from same position. Stream URL stubbed via `route()`.
