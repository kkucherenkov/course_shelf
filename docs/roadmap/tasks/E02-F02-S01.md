## Notes

- Completed: 2026-04-27 (retroactive flip via T-2026-04-27-030)
- Deviation: the card asked for `import { paths, components, createClient } from '@app/api-client-ts'` shaped around the manual `openapi-fetch` pattern. The actual generator is `@hey-api/openapi-ts` which produces a functional SDK with named operation functions (`recordLessonProgressBatch`, `getCourseDetail`, …) plus per-operation type aliases. Equivalent capability, more idiomatic for the generator chosen. Bearer auth is set on the global client config at the consumer side (frontend stores) rather than via a token-getter callback baked into a `createClient` factory.
- The generated package keeps a `'DO NOT EDIT BY HAND'` notice in `package.json#description`. The CLAUDE.md rule list explicitly forbids hand-editing `packages/api-client-ts/src/generated/`.
