- Decisions (see design doc): built on Material `ButtonStyleButton` +
  `WidgetStateProperty` with the ink ripple suppressed, so the flat web look is
  reproduced while focus/keyboard/semantics/disabled come for free; interaction
  is a flat token colour swap (`accentActive` on press), not a ripple. Label
  fonts bind off `Theme.textTheme.labelLarge` (packaged sans), never the bare
  `AppTextStyles` family that degrades to the platform font in consuming apps —
  a real fix caught by the golden. This is the shared recipe the E17-F01
  component wave reuses.
