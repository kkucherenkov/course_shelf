- Decisions (see design doc): rendered with **flutter_svg** from the web
  component's **verbatim** SVG markup rather than hand-drawn `CustomPainter`s —
  lowest-risk path to parity (no manual path/arc transcription). `currentColor`
  is resolved via a `srcIn` `ColorFilter` (`color` → `IconTheme` →
  `colorScheme.onSurface`); `fill` templates the `play`/`bookmark` geometry.
  Goldens are per-size grid sheets (`icon_grid_{16,20,24}` + `icon_grid_dark_24`,
  all 66 glyphs) instead of 198 per-icon PNGs. Registered in the mobile
  Widgetbook catalog (gallery / sizes / fill). The `pdf` glyph's `<text>` renders
  stably despite flutter_svg's limited text support (accepted — goldens are
  Flutter-vs-Flutter regression, not web pixel-diff).
