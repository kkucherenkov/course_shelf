## Goal

A simple markdown editor with toolbar, edit/preview toggle, and a "Saved · Ns ago" indicator (always inline, never modal).

## Acceptance

- Modes: edit, view
- Sync indicator states: Syncing, Saved, Failed (retrying), Offline (queued) — all small inline
- Auto-save debounced to 600ms after last keystroke
